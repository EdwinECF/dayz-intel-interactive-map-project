# dayz-intel-interactive-map-project

## About the Project

DayZ Intel Interactive Map is a web application designed to help DayZ players navigate and survive more efficiently.
The goal is to create a centralized platform that combines an interactive map, survival guides, and useful game information in one place.

This project is being built as both a useful tool for the DayZ community and a personal software engineering portfolio project.

---

## Vision

I wanted a tool that could help players without modifying the game or interacting with DayZ's files. 
The idea is to provide information and planning tools that improve the player's experience while respecting the game's rules and integrity.

---

## Planned Features

### Interactive Map

* Zoomable map of Chernarus and other supported maps.
* Custom player markers.
* Save favorite locations.
* Search for cities, military zones, and landmarks.
* Route planning between locations.

### Survival Guides

* Medical guides.
* Hunting and fishing guides.
* Base-building guides.
* Crafting recipes.
* Beginner tutorials and tips.

### Loot Information

* Loot tier maps.
* Military and civilian loot references.
* Item database.
* Spawn information.

### Community Features

* Share routes and markers.
* Create and save guides.
* Favorite locations and notes.
* Future community contributions.

### Personal Notes System

* Save private notes.
* Create personal objectives.
* Organize raid plans and base locations.
